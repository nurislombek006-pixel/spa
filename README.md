# Chess Game Review SPA

Одностраничное React/Vite-приложение для анализа PGN в стиле chess.com Game Review: доска, список ходов, классификации, график оценки, CAPS и отчёт.

## Запуск

```bash
npm install
npm run copy-engine
npm run dev
```

Открой адрес, который покажет Vite, обычно `http://localhost:5173`.

## Важно про Stockfish

Проект использует npm-пакет `stockfish` и копирует его wasm/js-файлы в `public/stockfish`. Если при запуске появится сообщение, что Stockfish не найден, выполни:

```bash
npm install stockfish
npm run copy-engine
```

Воркeр находится в `public/stockfish-worker.js`. Анализ идёт строго на клиенте, без сервера.

## Где менять глубину

В `src/main.jsx` в вызове `analyzeGame` стоит `depth: 18`.

## Где расширять дебютную книгу

Файл: `src/data/book.js`. Добавляй линии в UCI-формате, например:

```js
'e2e4 e7e5 g1f3 b8c6 f1b5 a7a6'
```

## Реализовано

- вставка PGN и загрузка `.pgn` файла;
- обрезка вариантов PGN;
- анализ первой партии;
- Stockfish Web Worker, последовательная очередь задач;
- depth 18;
- подсветка последнего хода;
- навигация и автопроигрывание 0.8 сек;
- классификации: 📖, !!, !, ★, —, ?!, ?, ??;
- win probability по формуле из ТЗ;
- CAPS отдельно за белых и чёрных;
- стадии: дебют, миттельшпиль, эндшпиль;
- график оценки;
- тренерские комментарии по шаблонам.

## Ограничение

Это приближение к chess.com, а не копия их закрытого Game Review. Точные chess.com-лейблы могут отличаться, потому что их эвристики и база дебютов закрыты.
